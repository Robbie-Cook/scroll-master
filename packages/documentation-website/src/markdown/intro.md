## Installation

### ES6 / CommonJS

```typescript
import ScrollMaster from "scroll-master";
```

or

```typescript
const { default: ScrollMaster } = require("scroll-master");
```

then, you can call scroll master to keep your item in view

```typescript
new ScrollMaster(".selector");
```
